Multi-Agent Crime Reporting Chatbot

This project is an AI-powered chatbot designed to help users report crimes easily and accurately. It can understand text, images, and voice input to identify the type of offence and generate structured crime reports.

The system uses multiple AI agents to analyze information, verify legal context, and produce clear, organized outputs that can be stored or shared.

Key Features

Text, image, and voice-based crime reporting

Automatic offence recognition using AI models

Legal context added to reports for clarity

Generates structured reports in PDF and CSV formats

Designed to be accessible and user-friendly
